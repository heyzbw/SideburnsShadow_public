## 鬓影（SideburnsShadow）

> `SideburnsShadow_public`是目前SideburnsShadow项目可以公开的备份仓库

目标实现对中国古代发型的生成

> HairClip for traditional Chinese hairstyle

项目主要工作分为
- 数据集收集制作
- e4e图像编码
- HairClip发型训练
- Cog环境下的推理预测

## 致谢
